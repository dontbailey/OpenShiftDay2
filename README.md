## Running Locally

1. Clone this github repo

```bash
git clone https://github.ibm.com/CE-National/TechDemos
```

2. Verify mkdocs and mkdocs material are installed.

```bash
npm install
```

3. Build the site

```bash
mkdocs build
```

4. Serve the Site

```bash
mkdocs serve
```
